# SafePix App Store pages

Support, Privacy Policy, and Marketing pages for App Store compliance.

## Use as standalone repo

1. Create a new GitHub repository named **safepix-site**.
2. Copy `index.md`, `privacy.md`, and `marketing.md` to the **root** of that repo.
3. Go to **Settings → Pages** → Source: **Deploy from branch** → Branch: **main** → Folder: **root**.
4. Your URLs will be:
   - **Support:** `https://YOUR_USERNAME.github.io/safepix-site/`
   - **Privacy:** `https://YOUR_USERNAME.github.io/safepix-site/privacy`
   - **Marketing:** `https://YOUR_USERNAME.github.io/safepix-site/marketing`
5. Enter these URLs in App Store Connect (Support URL, Privacy Policy URL, Marketing URL optional).

## App Store Connect fields

| Field | URL |
|-------|-----|
| Support URL | `https://YOUR_USERNAME.github.io/safepix-site/` |
| Marketing URL | `https://YOUR_USERNAME.github.io/safepix-site/marketing` (or leave empty) |
| Privacy Policy URL | `https://YOUR_USERNAME.github.io/safepix-site/privacy` |

Replace `YOUR_USERNAME` with your GitHub username.
